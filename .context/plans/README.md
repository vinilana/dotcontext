## Plan Lifecycle

1. **Proposed** -- Plan drafted, awaiting review.
2. **Approved** -- Reviewed and accepted for implementation.
3. **In Progress** -- Actively being worked on.
4. **Complete** -- Merged and verified.
5. **Abandoned** -- No longer relevant; kept for historical context.
